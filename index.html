Angular

What is Angular?

Angular is...

  • a front-end JavaScript framework

  • designed for cross-platform development

    (web, mobile web, native mobile, native desktop)

  • optimized for component-based MVC architecture

    MODEL: forest.service.ts

    VIEW: maple.component.html, maple.component.css

    CONTROLLER: maple.component.ts

Cordova

Cordova is...

  • a mobile app development framework

  • designed for cross-platform development

    (Android, iOS, Windows Phone)

  • based on platform-specific 'wrappers'

Things you can do with Cordova:

  • build hybrid mobile apps with HTML5, CSS, and JavaScript
  • deploy to multiple platforms (e.g., Android & iOS) with the same codebase
  • access native device APIs like geolocation sensors, cameras, altimeters, and more

Ionic is...

  • a UI framework for hybrid mobile apps

    "kind of like 'Bootstrap for Native'"

  • an HTML5 framework that requires a native wrapper (such as Cordova)
  • built on top of Cordova and Angular

Question 1

True or false:

Angular is used exclusively for mobile app development

FALSE

Angular can be used for web and desktop applications as well as for mobile web and native (or hybrid) mobile apps.

Question 2

a) A JavaScript framework

b) A UI framework

c) Something to do with chemistry?

B

Question 3

True or false:

Ionic requires both Angular and Cordova in order to be fully functional

TRUE

Many of Ionic's UI components are built with Angular, and Ionic apps rely on Cordova in order to run as native apps.

Question 4

Which of the following can you do with the help of Cordova?

a) Build an iOS app without writing a single line of Swift or Objective-C

b) Access mobile device sensors in order to display a user's current altitude

c) Deploy an app to Android with the same code you used for an iOS app

All of the above!

So what?

Angular helps you write tightly knit, client-side MVC components, which you can polish up with the mobile-optimized design elements of Ionic. Wrap it all up with Cordova, and your app will feel at home on just about any mobile device.

In other words, these three tools make it easier than ever to build nearly-native mobile apps with the web development languages you're used to (i.e., HTML, CSS, and JavaScript).
